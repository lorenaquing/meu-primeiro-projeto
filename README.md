# meu-primeiro-projeto
Essa atividade me ajudou a compreender melhor o que são o Git e o GitHub e como eles são fundamentais no desenvolvimento de software. Aprendi que o Git é um sistema de controle de versão que guarda o histórico de tudo o que foi alterado em um projeto. Isso é muito útil para acompanhar o progresso e evitar a perda de informações. Já o GitHub é uma plataforma que funciona junto com o Git, permitindo armazenar os projetos online e trabalhar em equipe com outras pessoas.

No início, achei um pouco confuso entender os comandos, mas depois que comecei a praticar ficou mais fácil. Criar o repositório e enviar arquivos para ele foi um passo importante, e gostei de ver meu primeiro projeto publicado no GitHub. Mesmo usando o celular, consegui entender bem o funcionamento das etapas e perceber a utilidade dessas ferramentas.

Em resumo, foi uma experiência positiva e produtiva. Aprender sobre Git e GitHub me fez perceber que eles são essenciais para quem quer trabalhar com tecnologia e desenvolvimento, pois trazem organização, segurança e colaboração aos projetos.
